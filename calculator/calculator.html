    CLI Calculator

    In this project, you'll learn various ways in which you can use Python as a calculator from the command line. You'll also learn how you can customize the features for your specific needs. First, you'll see how you can directly pass Python code from the command line and create bash shortcuts to simplify the invocation. Second, you'll see how to use Python features to create a custom CLI application. Finally, you'll be given exercises to test your understanding and resource links for further exploration. The links to these sections are given below:

    • Bash shortcuts
    • Python CLI application
    • Exercises

    Project summary

    • Execute Python instructions from the command line
    • Use shell shortcuts to simplify command line typing
    • Evaluate string content as Python code
    • Create user friendly command line interfaces
    • Allow stdin as source of user input
